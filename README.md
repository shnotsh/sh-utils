# SH Utils

Personal Godot Utilities plugin.

### Shortcuts
Includes simpler keyboard shortcuts:

| Key | Action |
| --- | --- |
| `Esc` | Force Quit Application |
| `~` (Tilde) | Toggle Debug Overlays |
| `V` | Toggle VSync |
| `F` | Toggle Fullscreen |

Keyboard shortcuts works only when debug overlays are visible.

## Installation

1. Copy the `addons/sh_utils` folder to your project's `addons/` directory.
2. Go to **Project > Project Settings > Plugins**.
3. Enable **SH Utils**.

The plugin will automatically add a `Debug` autoload singleton.
