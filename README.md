# SH Utils

Personal Godot Utilities plugin.

### Shortcuts
| Key | Action |
| --- | --- |
| `~` | Toggle Debug Overlays |
| `Esc` | Force Quit Application |
| `V` | Toggle VSync |
| `F` | Toggle Fullscreen |

Keyboard shortcuts works only when debug overlays are visible.

## Installation

1. Copy the `/sh_utils` folder to your project's `addons/` directory.
2. Go to **Project > Project Settings > Plugins**.
3. Enable **SH Utils**.

The plugin will automatically add a `Debug` and `Utils` autoload singletons.
